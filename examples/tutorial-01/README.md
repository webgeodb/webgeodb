## 📖 核心概念

### 1. 数据库（Database）

WebGeoDB 数据库是地理空间数据的容器，存储在浏览器的 IndexedDB 中（浏览器环境）或文件系统中（Node.js 环境）。

### 2. 表（Table）

表是存储特定类型数据的集合。例如，`features` 表用于存储地理要素。

### 3. 索引（Index）

索引用于加速查询：
- **属性索引**: 基于字段值的标准索引
- **空间索引**: 基于 R-tree 的空间索引，用于加速空间查询

### 4. GeoJSON

WebGeoDB 使用 GeoJSON 格式存储地理空间数据：
- **Point**: 点（如商店位置）
- **LineString**: 线（如道路）
- **Polygon**: 面（如公园、建筑物）
