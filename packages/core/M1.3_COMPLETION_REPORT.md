## 完成内容

### 1. 错误类型定义（`src/errors/index.ts`）

#### 基础错误类
- **WebGeoDBError**: 基类，包含：
  - 错误代码（ErrorCode 枚举）
  - 严重级别（ErrorSeverity: LOW/MEDIUM/HIGH/CRITICAL）
  - 错误上下文（ErrorContext）
  - 时间戳
  - 堆栈跟踪
  - `getDetails()` 方法：获取详细的错误信息
  - `toJSON()` 方法：序列化为 JSON

#### 专用错误子类（8 个）
1. **DatabaseError** - 数据库错误（HIGH 级别）
   - DATABASE_NOT_OPEN
   - DATABASE_ALREADY_OPEN
   - DATABASE_CLOSED
   - DATABASE_VERSION_MISMATCH
   - DATABASE_INIT_FAILED

2. **TableError** - 表错误（MEDIUM 级别）
   - TABLE_NOT_FOUND
   - TABLE_ALREADY_EXISTS
   - TABLE_SCHEMA_INVALID

3. **QueryError** - 查询错误（MEDIUM 级别）
   - QUERY_INVALID
   - QUERY_PARSE_FAILED
   - QUERY_EXECUTION_FAILED
   - QUERY_TIMEOUT

4. **ValidationError** - 验证错误（LOW 级别）
   - VALIDATION_FAILED
   - INVALID_GEOMETRY
   - INVALID_PARAMETER
   - INVALID_SCHEMA

5. **TransactionError** - 事务错误（HIGH 级别）
   - TRANSACTION_FAILED
   - TRANSACTION_ABORTED
   - TRANSACTION_CONFLICT

6. **IndexError** - 索引错误（MEDIUM 级别）
   - INDEX_NOT_FOUND
   - INDEX_ALREADY_EXISTS
   - INDEX_CREATE_FAILED
   - INDEX_CORRUPTED

7. **StorageError** - 存储错误（HIGH 级别）
   - STORAGE_ERROR
   - STORAGE_QUOTA_EXCEEDED
   - STORAGE_NOT_AVAILABLE
   - STORAGE_WRITE_FAILED

8. **SQLError** - SQL 错误（MEDIUM 级别）
   - SQL_SYNTAX_ERROR
   - SQL_NOT_SUPPORTED
   - SQL_PARAMETER_MISSING
   - SQL_FUNCTION_NOT_FOUND

9. **SpatialError** - 空间错误（MEDIUM 级别）
   - SPATIAL_ENGINE_NOT_FOUND
   - SPATIAL_OPERATION_FAILED
   - SPATIAL_INVALID_COORDINATE

  console.log('错误代码:', error.code);

## 使用示例

### 基本使用
```typescript
import { WebGeoDB } from 'webgeodb';
import { ErrorFactory, DatabaseError, TableError } from 'webgeodb/errors';

const db = new WebGeoDB({ name: 'mydb', version: 1 });

try {
  await db.open();
  // ... 数据库操作
} catch (error) {
  if (error instanceof DatabaseError) {
    console.error(`数据库错误 [${error.code}]: ${error.message}`);
    console.error('详细信息:', error.getDetails());
  } else if (error instanceof TableError) {
    console.error('表操作失败:', error.message);
  }
}
```

### 使用 ErrorHandler
```typescript
import { ErrorHandler } from 'webgeodb/errors';

const result = await ErrorHandler.handleAsync(async () => {
  const db = new WebGeoDB({ name: 'mydb', version: 1 });
  await db.open();
  return db;
});
```

### 创建自定义错误
```typescript
import { ErrorFactory } from 'webgeodb/errors';

const error = ErrorFactory.tableNotFound('users');
console.log(error.getDetails());
// [TABLE_NOT_FOUND] Table 'users' not found in schema.
//   Context: { tableName: users }
```
