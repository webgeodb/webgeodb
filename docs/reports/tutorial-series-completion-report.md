## 📖 使用指南

### 学习路径建议

1. **初学者**
   - 从第1章开始，按顺序学习
   - 运行所有示例代码
   - 完成实战练习

2. **有基础的开发者**
   - 跳过第1章，从第2章开始
   - 关注高级特性和优化
   - 研究专题应用

3. **架构师/高级开发者**
   - 直接学习第5章
   - 研究专题应用代码
   - 参考架构设计

### 快速开始

```bash
# 克隆项目
git clone https://github.com/zhangyuting/webgeodb.git

# 安装依赖
pnpm install

# 运行示例
cd examples/tutorial-01/01-first-database
pnpm start

# 查看演示
cd examples/tutorial-02/demos
python -m http.server 8001
# 访问 http://localhost:8001
```
